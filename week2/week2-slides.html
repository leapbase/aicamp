Session 1: Practical Prompting Techniques

1. Be Specific and Direct

Vague prompts get vague results. Specificity eliminates ambiguity. Example: "Write a Python function with type hints that filters a list of dictionaries by status field"

2. Provide Examples (Few-Shot Prompting)

Show format and edge cases through examples. This demonstrates what you want and handles corner cases

3. Assign a Role (Role Prompting)

"You are an expert Python developer reviewing code for security vulnerabilities" - activates relevant training data

4. Use Chain-of-Thought

"Think through the problem step by step before answering" - improves reasoning on complex problems

System Prompts: The Hidden Layer

What System Prompts Control

  • Personality and tone
  • Expertise level
  • Output format
  • Safety guardrails
  • Consistent behavior

Best Practices

  • Set once at conversation start
  • Be clear and specific
  • Define boundaries
  • Include output format requirements
  • Explain the "why" behind decisions

Think of System Prompts as Configuration

System prompts are like the "rules of engagement" for an AI conversation. They influence every response without being repeated in each turn.

Why Structured Output Matters

The Problem

  • Free-form text is hard to parse
  • Format varies response-to-response
  • Fragile integration with apps
  • Difficult to validate

The Solution

  • JSON Mode: Forces valid JSON
  • Function Calling: Type-safe parameters
  • Reliable parsing
  • Easy integration

Structured Output Bridges the Gap

From demos to production apps - JSON mode and function calling make AI output reliable and parseable

6 / 12

Function Calling / Tool Use

The Concept

1. Define function schema (name, description, parameters)
2. Model decides when to call your function
3. Model generates function call with arguments
4. Your code executes and returns results
5. Model uses results to continue conversation

Function Calling Gives LLMs Superpowers

They can search databases, call APIs, do calculations, and more - all by invoking functions you control. This is the foundation of AI agents.
