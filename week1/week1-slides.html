Session 1: Environment Setup

Tools to Install

  • Python 3.10+
  • VS Code + Extensions
  • Git & GitHub
  • Virtual environments

API Setup

  • OpenAI account & API key
  • Anthropic account & API key
  • Set billing caps ($10 to start)
  • Environment variables

Security Best Practice

Never hardcode API keys! Use .env files with python-dotenv. Add .env to .gitignore immediately.

5 / 12

Your AI Development Environment

1. Python 3.10+

The primary language for AI development. Install via python.org or package managers. Verify with python --version

2. VS Code

Your code editor with Python, Pylance, and Jupyter extensions installed

3. Git & GitHub

Version control is essential. Configure your identity and create a GitHub account

4. Virtual Environments

Keep dependencies isolated with python -m venv myenv

Applying the 4 D's to Your Work

Delegation

What programming tasks can AI handle? Which should you always do yourself?

Description

How is writing a prompt similar to writing a function docstring?

Discernment

How will you verify that AI-generated code actually works and is secure?

Diligence

When AI gives you wrong code, what's your debugging process?

No single right answers - these are frameworks for thinking about your work

Session 2: Making Your First API Calls

OpenAI SDK

  • Install: pip install openai
  • Set up authentication
  • Create chat completion with GPT-4
  • Parse response
  • Understand tokens and pricing

Anthropic SDK

  • Install: pip install anthropic
  • Compare API structure to OpenAI
  • Use Claude Sonnet or Opus
  • Note parameter differences
  • Compare response formats

Hands-On Exercise

Build a script that sends input to both APIs and displays responses side-by-side with token counts
