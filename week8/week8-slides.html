What Is an Agent?

Definition

An agent is an AI system that can perceive its environment, make decisions, and take actions to achieve goals - autonomously and iteratively.

Agents vs. Tools

  • Tool: Human-in-the-loop
  • Step-by-step guidance
  • Requires approval
  • Example: Claude Code

Agent Characteristics

  • Agent: Autonomous
  • Multi-step execution
  • Minimal human intervention
  • Goal-driven behavior
3 / 12

The Think-Act-Observe Loop

🧠

Think

Analyze current state and decide what to do next

⚡

Act

Execute an action: call API, run command, write file

👁️

Observe

See the results and update understanding

The Loop Continues

Repeat the cycle until the goal is achieved or a stopping condition is met. This pattern is fundamental to all agent architectures.

Types of Agents

ReAct (Reason + Act)

Explains reasoning before each action. Transparent decision-making.

Plan-and-Execute

Creates full plan upfront, then executes step-by-step.

Reflexive

Reacts to environment without explicit planning. Fast responses.

Hierarchical

Multiple agents with different roles working together.

Real-world examples: Code debugging bots, customer support automation, research assistants, DevOps automation, personal assistants

Common Agent Failures & Debugging

Failure Modes

  • Looping: Repeats same action
  • Hallucination: Claims false actions
  • Tool Misuse: Wrong tool/args
  • Premature Completion
  • Excessive Steps

Debugging Techniques

  • Log every LLM call
  • Add step numbers
  • Print agent reasoning
  • Inspect failing cases
  • Add assertions

Improvement Ideas

Add reflection step, implement memory, track costs, better stopping conditions

Agent Security & Blast Radius

Critical Security Principles

  • Least Privilege: Grant minimal permissions needed
  • Human-in-the-Loop: Require approval for destructive actions
  • Audit Logging: Log every action for accountability
  • Rate Limiting: Prevent runaway loops
  • Sandboxing: Isolate agent execution

Think About Blast Radius

Always ask: "If this agent were fully compromised, what's the worst that could happen?" Design permissions accordingly.
